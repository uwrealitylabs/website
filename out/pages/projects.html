Reality From Scratch

By using an IMU connected to an Arduino Pro Micro, we achieved rotational (3DoF) tracking. The IMU motion vector data is translated into SteamVR readable input through drivers forked from the OpenVR SDK.

Project Northstar

A fully self-contained AR device using optical combiners and Intel's RealSense T265. Based on Ultraleap's Project North Star.
